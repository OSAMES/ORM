ConfigurationLoader ClassOsames Micro ORM
This class is dedicated to reading configuration files: app.config and specific files in Config subdirectory. Then data is loaded into internal directories. There are also overloads to allow configuration bypass, for example in unit tests where it's easier to call an overload to use another XML template file.
