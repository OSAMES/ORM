DbTools FormatSqlForSelect Method (String, String, List String , List Object , List String , String , List KeyValuePair String, Object   )Osames Micro ORM
Crée le texte de la commande SQL paramétrée ainsi que les paramètres ADO.NET, dans le cas d'un select basé sur un template "SELECT * FROM {0}...". Utilise : - clé du dictionnaire de mapping - liste de noms de propriétés de dataObject_ à utiliser pour les champs à mettre à jour - liste de noms de propriétés de dataObject_ ou paramètres dynamiques pour les paramètres dans la partie WHERE.
